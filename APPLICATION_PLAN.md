# LeadGen AI - Application Plan

LeadGen AI is a modern AI-powered SaaS platform designed for B2B lead generation and outreach automation.

## Core Features (MVP)

### 1. Lead Discovery System
- **Advanced Search**: Filter leads by Industry, Target Role, Company Size, and Location.
- **AI-Powered Results**: Simulates intelligent discovery of potential B2B companies.
- **Detailed Data**: Each lead includes Company Name, Industry, Website, Role, and Contact info.

### 2. AI Lead Enrichment
- **One-Click Analysis**: Generate deep company insights using simulated AI.
- **Pain Points**: Identify 3-4 specific business problems the target might face.
- **Sales Angle**: Personalized strategic advice for the outreach approach.

### 3. AI Outreach Automation
- **Cold Email Generator**: Professional, friendly, and persuasive emails under 120 words.
- **LinkedIn Message Generator**: Concise, natural-sounding DMs under 60 words.
- **Copy & Manage**: Easy copying of generated content to clipboard.

### 4. Lead Dashboard & Management
- **Centralized CRM**: Save leads from search results to a persistent dashboard.
- **Usage Tracking**: Monitor how many emails and messages have been generated per lead.
- **CSV Export**: Export all saved leads and their data for use in other CRM tools.

## Design System

- **Colors**: Primary Blue (#2b7fff), Clean White Background, Slate/Zinc Gray for secondary elements.
- **Typography**: Poppins/Sans-serif for a modern, clean look.
- **UI Components**: 
  - Rounded cards (1rem radius)
  - Subtle shadows for depth
  - Animated Side Drawer for lead details (Framer Motion)
  - Interactive tables with hover states
- **Icons**: Lucide React for consistent, minimal iconography.

## Tech Stack

- **Frontend**: React 19, Vite, TypeScript.
- **Styling**: Tailwind CSS v4.
- **Routing**: React Router v7.
- **Animations**: Framer Motion.
- **Storage**: Local Storage for persisting saved leads and generated content.
